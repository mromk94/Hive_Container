# OMK Container Code of Conduct

OMK Container is part of the AI-Verse / Hive ecosystem and inherits its safety-first philosophy.

- Treat all collaborators with respect.
- No harassment, hate speech, or personal attacks.
- Assume good intent; disagree on ideas, not on people.
- Protect user privacy and follow the spirit of the Shared Consciousness Loop: humans first, systems second.

Reports can be sent privately to the maintainer email listed in the main README.
